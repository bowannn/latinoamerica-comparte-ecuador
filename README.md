# Ecuador Comparte - Sistema Integrador

Backend robusto desarrollado en **Java con Spring Boot** para la plataforma "Ecuador Comparte", encargado de la gestión de noticias, testimonios, autenticación de usuarios e inicialización automática de datos para la gestión comunitaria e informativa.

## Arquitectura y Tecnologías Utilizadas

El proyecto está construido bajo una arquitectura limpia en capas (Controller, Service, Repository, Entity) utilizando el siguiente stack tecnológico:

* **Lenguaje:** Java 21
* **Framework:** Spring Boot 4.0.6 (Spring Data JPA, Spring Security)
* **Base de Datos:** PostgreSQL 18
* **Gestor de Dependencias:** Maven
* **Contenedores:** Docker (para empaquetado y despliegue)
* **Servidor de Aplicaciones:** Apache Tomcat 11 integrado
